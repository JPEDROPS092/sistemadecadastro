# Sistema de Gestão de Estoque com Flask

Sistema web para controle de estoque, entrada e saída de produtos, com relatórios diários e semanais de lucro, usando Python, Flask e SQLite.

---

## Funcionalidades

- Cadastro de produtos com quantidade, valor de compra e venda.
- Registro de entradas e saídas de produtos.
- Histórico completo de movimentos com data e hora.
- Relatórios diários e semanais de lucro líquido.
- Gráficos de lucro diário e semanal usando Chart.js.
- Limpeza total do estoque.

---

## Tecnologias

- Python 3
- Flask
- SQLAlchemy (ORM)
- SQLite (banco de dados)
- HTML/CSS
- Chart.js (gráficos)
